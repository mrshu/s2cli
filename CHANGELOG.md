## [0.1.1] - 2026-01-23

### Added

- `uvx` usage example in README for running without installation

### Changed

- Updated package metadata (author, repository URLs)

## [0.1.0] - 2026-01-15

### Added

- Initial release
- Search papers by keyword with filters (year, citations, open access, venue)
- Get paper details with batch support
- Get citations and references for papers
- Paper recommendations
- BibTeX export
- Author search and paper listing
- Dataset listing
- Human-readable tables in terminal, automatic JSON when piped
- `--json` and `--bibtex` output flags
- Automatic retry with countdown on rate limits
- Smart TTY detection for output formatting
